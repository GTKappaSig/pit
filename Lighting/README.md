# Pit Lighting
## Instructions for adding scenes
1. Hold down the program button for three seconds
2. Click the fixture button for the fixture you want (example Fixture 1)
3. Set the light how you want it using the different channels
4. Continue this process for the other fixtures
5. Click rec
6. Click the scene number (example 1)
## Important RGB colors
- Red = 255, 0, 0
- Green = 0, 255, 0
- Blue = 0, 0, 255
- White = 255, 255, 255
- Orange = 255, 128, 0
- Yellow = 255, 255, 0
- cyan = 0, 255, 255
- purple = 128, 0, 255
- magenta = 255, 0, 255
- Georgia tech gold = 179, 163, 105
- Gold I used = 218, 165, 32
- Georgia tech blue = 0, 38, 58
- Kappa Sigma Red = 191, 13, 62
- Kappa Sigma Green = 201, 151, 0
